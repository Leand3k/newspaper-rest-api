# Newspaper REST API

This API was made for a web development class from INTEC.

It has a few routes, including:

Files:
```
/files
/files/<int:idFile>
/files/delete/<int:idFile>
```

Articles:
```
/article/create
/article/<int:idArticle>
/article/all
/article/delete/<int:idArticle>
/article/edit/<int:idArticle>
```

Admin:
```
/admin/create
/admin/<int:idAdmin>
/admin/delete
/admin/edit
/login
/logout
```
